## Usage
To use this application you have to download [Node](https://nodejs.org/en/), install de node.js and open the terminal on the current folder where you download theses files and run these command below.


```sh
npm init -y
```

```sh
 npm install express 
```

```sh
 npm body-parser
```

```sh
 npm mongoose
```

```sh
 npm bcryptjs
```
